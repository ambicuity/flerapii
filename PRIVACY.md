# Flerapii Privacy Policy

Last updated: March 16, 2026

This Privacy Policy applies to the Flerapii browser extension.

## Summary

- Flerapii is local-first. It does not operate a telemetry backend and does not sell personal information.
- Extension data is stored in browser extension storage on your device.
- Optional sync features only send data to services you configure (for example your own WebDAV server or API relay endpoints).
- You control data by editing/removing accounts in the extension or clearing browser extension storage.

## Data We Process

Flerapii processes the following categories of data to deliver extension features:

- Account configuration data you enter (site URL, account labels, API keys/tokens, optional notes/tags)
- Operational metadata generated by extension features (check-in status, refresh state, usage history, balance snapshots, model/channel preferences)
- Optional integration settings (WebDAV endpoint, credentials, and backup preferences when enabled)

Flerapii does not intentionally collect personal profile data beyond the values you provide to manage your API relay accounts.

## How Data Is Used

Data is used only to provide requested extension functionality, such as:

- Account management and key/model operations
- Auto refresh, auto check-in, and redemption assistance
- Import/export and optional sync/backup flows
- UI state, preferences, and diagnostics controlled by you

Data is not used for advertising, cross-site tracking, or profile monetization.

## Where Data Is Stored

By default, data is stored in browser extension storage on your device.

If you enable optional sync/backup features, data may be transmitted to destinations you configure, such as:

- Your API relay endpoints
- Your WebDAV server

Flerapii does not provide a first-party cloud storage service for your extension data.

## Data Sharing

Flerapii does not share your data with third parties except when you explicitly configure and trigger integrations. Communication only occurs with endpoints required by features you enable.

## Data Retention and Deletion

Data remains available until you remove it.

You can delete data by:

- Removing accounts/bookmarks/settings in the extension UI
- Importing replacement data
- Clearing extension storage via browser extension management
- Disabling/removing optional sync integrations

## Security

The project applies reasonable safeguards in code and workflow, but no client-side system is absolutely secure.

You are responsible for keeping endpoint credentials and exported backup files secure.

## Children's Privacy

Flerapii is not intended for children under 13 and is not designed to knowingly collect data from children.

## Regional Rights

Depending on your jurisdiction, you may have rights to access, correction, deletion, or portability of your data. Because Flerapii is local-first and user-controlled, these operations can typically be performed directly in your browser environment.

## Policy Changes

This policy may be updated when product behavior changes. Material updates will be reflected by updating the date above and repository documentation.
