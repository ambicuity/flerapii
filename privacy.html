Flerapii Privacy Policy

Last updated: March 16, 2026

Overview

Flerapii is a local-first browser extension for managing AI relay account connections, keys, models, and related workflows.

This policy explains what data the extension processes, where data is stored, and what controls you have.

Data We Process

Flerapii processes data that you provide or that is generated by extension features, including:

  • Site/account configuration (for example endpoint URLs, labels, and API credentials)
  • Feature metadata (for example check-in state, refresh status, usage history, and balance snapshots)
  • Preferences and UI settings
  • Optional sync/backup settings (for example WebDAV endpoint settings)

Flerapii does not provide a first-party telemetry backend and does not sell personal data.

How Data Is Used

Data is used only to deliver extension functionality you choose, such as:

  • Account management
  • Auto refresh and auto check-in
  • Import/export and backup
  • Model and channel management
  • In-extension diagnostics controlled by your settings

Data is not used for advertising, cross-site tracking, or behavioral profiling.

Data Storage and Transmission

Default storage

By default, extension data is stored in browser extension storage on your device.

Optional user-configured transmission

If you enable optional integrations, Flerapii may send data to destinations you configure, such as:

  • API relay endpoints you add
  • Your WebDAV server

These flows are user-configured and feature-driven. Flerapii does not operate a managed cloud database for your extension data.

Permissions and Limited Access

Some optional browser permissions (for example cookie and web-request permissions) are used only for requested flows such as protection bypass and authenticated refresh.

These permissions are revocable in browser settings or from the extension UI.

Third Parties

Flerapii does not embed third-party analytics or ad trackers.

Third-party network interactions occur only with endpoints required by features you explicitly configure and use.

Retention and Deletion

Data is retained until removed by you.

You can remove data by:

  • Editing/deleting entries in extension UI
  • Importing replacement data
  • Clearing extension storage in browser extension settings
  • Disabling optional sync or removing extension installation

Security Considerations

Reasonable safeguards are applied in code and workflow, but no local or networked system can guarantee absolute security.

You are responsible for protecting exported backup files and credentials in your own environment.

Children's Privacy

Flerapii is not intended for children under 13 and is not designed to knowingly collect data from children.

Regional Privacy Rights

Depending on jurisdiction, you may have rights related to access, correction, deletion, or portability.

Because Flerapii is local-first and user-controlled, these operations can generally be performed directly in your browser storage and configuration.

Policy Changes

This policy may be updated as product behavior evolves. Changes are reflected by updating the date at the top of this page.

Contact

For privacy questions, open an issue in the repository:

  • https://github.com/ambicuity/flerapii/issues
